# 마크다운
## 마크다운(Markdown)이란?
2004년에 존 그루버(John Gruber)와 애런 스워츠(Aaron Swartz)가 만든 마
크업 언어의 하나로 읽기 쉽고 쓰기 쉬운 텍스트 포맷입니다.
이 페이지의 왼쪽은 마크다운 편집기입니다. **자유롭게 연습해 보세요**. 여러분이
연습한 내용은 다른 사람에게 보이지 않고, 저장되지 않습니다.
아래 링크를 클릭하여 해당 도움말(연습장)로 바로 이동할 수도 있습니다.
